# 图标 Icon

语义化的矢量图标。

## 代码演示
```javascript
// main.js

import { Icon } from 'news-ui-mobile'

Vue.use(Icon)
```

```html
<!-- *.vue -->

<template>
  <n-icon type="icon-forward"></n-icon>
</template>
```
## API

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| type | icon类型 | string | |

## 使用

点击复制图标类别。
